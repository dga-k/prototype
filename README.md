# prototype

โปรเจค HTML + JavaScript + Tailwind CSS — แอปรายการสิ่งที่ต้องทำ (To-Do List)

## วิธีใช้งาน

เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งหรือ build ใดๆ

## ฟีเจอร์

- ✅ เพิ่ม / ลบรายการ
- ✅ ติ๊กรายการที่เสร็จแล้ว
- ✅ กรองรายการ: ทั้งหมด / ยังไม่เสร็จ / เสร็จแล้ว
- ✅ ลบรายการที่เสร็จแล้วทั้งหมดในครั้งเดียว
- ✅ บันทึกข้อมูลใน `localStorage` (ข้อมูลไม่หายเมื่อปิดเบราว์เซอร์)

## เทคโนโลยีที่ใช้

- **HTML5**
